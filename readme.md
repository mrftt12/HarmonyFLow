# HarmonyStream

A premium music and video streaming app that combines the best of Spotify and YouTube. Discover, play, and enjoy your favorite songs and videos in one seamless experience.

## Style Guide

### Theme and Color Scheme
- **Primary Color**: #FF69B4 (Hot Pink)
- **Secondary Color**: #8B0A1A (Dark Red)
- **Accent Color**: #FFC67D (Light Orange)
- **Background**: #121212 (Almost Black)
- **Surface**: #1A1A1A (Dark Gray)
- **Text**: #FFFFFF (White)
- **Text Secondary**: #AAAAAA (Light Gray)

### Visual Design
- **Style**: Elegant and modern
- **Mood**: Bold and energetic
- **Visual Weight**: Heavy with rich gradients and shadows
- **Border Radius**: rounded-xl for all components
- **Padding**: p-6 for sections
- **Margin**: m-4 for spacing

## Features

- **Dual Streaming**: Seamlessly switch between music tracks and video content
- **Personalized Home**: Recently played content and recommendations
- **Music Discovery**: Browse popular tracks with album art
- **Video Library**: Access trending videos with thumbnails
- **Unified Player**: Control both music and video from one player bar
- **Navigation**: Intuitive bottom navigation for easy access

## Technologies Used

- React Native
- Expo
- NativeWind (Tailwind CSS for React Native)
- React Navigation
- Lucide Icons

## Installation Instructions

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd harmonystream
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

## Usage Instructions

- Browse music and videos from the home screen
- Tap on any track or video to play it
- Use the player bar at the bottom to control playback
- Navigate between sections using the bottom navigation
- Search for specific content using the search icon

## Design Principles

This app follows a dark theme design with vibrant accent colors to create a premium entertainment experience. The interface is optimized for both music and video consumption with large visual elements and intuitive controls.
